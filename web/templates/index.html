¿Qué es Urban Mobility Manager?

Urban Mobility Manager es una aplicación web que permite administrar datos de transporte público en diferentes ciudades. La aplicación proporciona información recopilada desde el simulador SUMO sobre las líneas de autobús, metro y tren de cercanías, así como las paradas de transporte público de la ciudad. Además, ofrece la posibilidad de editar rutas y relanzar simulaciones.
